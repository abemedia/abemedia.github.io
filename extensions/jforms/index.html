Description

donJoomla JForms Module is a small but powerful extension which builds AJAX forms from XML using Joomla's inbuilt JForm class.

It can be used as a basic AJAX contact form, but, being easily extensible with plugins, it could be used as a sign-up form, questionnaire or any form of data collection you can imagine.

Features

  • Create custom forms using XML
  • AJAX form submission
  • Send the form data in customised emails
  • Extensible with plugins
  • Uses Joomla! core validation
  • Supports Bootstrap 2 & 3
